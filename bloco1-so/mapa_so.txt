# PARTE B — Questão de reflexão
# A apresentação mostrou que o supervisor controlava o acesso a I/O
# e que rotinas de dispositivo traduziam comandos.
# Em qual diretório do Linux você reconhece essa função?

Resposta:
Eu reconheço essa função no diretório /dev, porque é lá que ficam os arquivos que representam os dispositivos de entrada e saída do sistema.
Pelo que entendi, esses arquivos funcionam como uma forma do sistema operacional se comunicar com o hardware,
parecido com as rotinas de dispositivo que traduziam comandos antigamente.
